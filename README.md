# k-usedcar

한국 중고차 사이트(엔카·케이카·기아 인증중고차·헤이딜러)에서 **내 조건에 맞는 매물을
조회·비교**하도록 돕는 AI 에이전트용 스킬 모음입니다. `k-skill` 스타일로, 헤드리스
브라우저(예: gstack `browse`)를 통해 각 사이트가 사용하는 공개 페이지/내부 조회
엔드포인트를 호출하고, 옵션·사고이력·시세를 구조화해 정리합니다.

> ⚠️ **읽고 시작하세요:** [DISCLAIMER.md](./DISCLAIMER.md) — 개인·비상업 용도 전용,
> 약관/robots.txt 준수·개인정보 처리 책임은 이용자에게 있습니다. **이 저장소에는
> 어떤 수집 데이터도 포함되지 않습니다(코드만 제공).**

## 무엇을 하나
- 모델·연식·주행거리·가격·연료·색상 등 **검색 필터**로 매물 목록 조회
- 매물 상세에서 **필수 옵션(후방카메라·대화면 내비·ADAS·통풍시트·서라운드뷰),
  선루프·HUD, 사고/보험이력, 렌트(용도변경) 여부, 변속기(DCT/자동)** 추출
- 사이트 간 결과를 **표로 비교**하고 추천 정리

## 구성
```
skills/
  encar/      # 엔카 (검색 API + readside 상세 API)
  kcar/       # 케이카 (car-info-detail API / 렌더 카드)
  kia-cpo/    # 기아 인증중고차 (api/search, product 상세)
  heydealer/  # 헤이딜러 (마켓 카드 + 상세 텍스트 파싱)
```
각 폴더의 `SKILL.md`에 호출 방법과 추출 로직, 주의사항이 있습니다.

## 사전 요구
- 헤드리스/헤드 브라우저 제어 도구 (예: [gstack `browse`](https://garryslist.org) 또는
  Playwright). 스킬은 "페이지 이동 → JS 평가로 데이터 추출" 패턴을 사용합니다.
- 일부 사이트는 CORS·가상스크롤·로그인 게이트·안티봇으로 **자동 수집이 제한**됩니다.
  각 `SKILL.md`의 "한계" 절을 참고하세요.

## 사용 원칙 (요약)
- 개인 구매 검토용 **소량·저빈도** 조회
- 요청 사이 지연으로 서버 부하 최소화
- **개인정보(딜러 연락처·차량번호 등)·수집 데이터는 저장/공유/재배포 금지**
