# 중고차 종합 비교·추천 HTML 생성 스킬

수집 스킬(encar/kcar/kia-cpo/heydealer)의 정규화 결과 배열을 받아, 단일 `index.html`
(무의존·정적)로 **TOP 3 추천 카드 + 전체 비교표 + 방법론 노트**를 만든다.

## 입력 데이터 스키마 (사이트 공통 정규화)
각 매물 객체. **개인정보 금지 필드는 받지도 말 것**(PRIVACY.md). 매물 링크는 공개
상세 URL만.
```json
{
  "site": "엔카|케이카|기아CPO|헤이딜러|현대인증",
  "model": "투싼 NX4 인스퍼레이션",
  "trim": "인스퍼레이션",
  "color": "흰색",
  "price_manwon": 2499,
  "year": "2022.06",
  "km": 30760,
  "fuel": "가솔린|디젤|하이브리드",
  "transmission": "7DCT건식|8DCT습식|8AT|6AT(HEV)",
  "dct": "건식7|습식8|무DCT",
  "options": {"후방카메라":true,"내비":true,"ADAS":true,"통풍시트":true,"서라운드뷰":true,"선루프":false,"HUD":false},
  "accident": "무사고|단순교환무사고|사고",
  "insurance": "전기간가입|미가입N구간|내차Nmanwon/타차Nmanwon",
  "rental": "없음|렌트|확인필요",
  "url": "https://...(공개 상세 페이지)"
}
```

## 생성 규칙
- **필수옵션 5종**(후방카메라·대화면내비·ADAS·통풍시트·서라운드뷰) 충족 여부 표기.
- **DCT 경고 박스**: 건식7DCT(가솔린1.6T)=저속 울컥 심함, 습식8DCT(디젤)=완화,
  무DCT(가솔린2.0/2.5·HEV)=무관.
- **흰색 우선 정렬**, 렌트/보험 미가입은 별도 구간으로 분리.
- TOP 3는 색상·이력·DCT·옵션 종합 점수로 선정하되 근거를 카드에 명시.
- **랭킹 가중치(권장)**: 패널티(낮을수록 상위) = `DCT위험*3 + (HEV면 0 아니면 가점차) +
  내차피해(백만원) - 옵션수(SVM·선루프)`. "비슷한 가격이면 하이브리드 선호"는 HEV에
  가점을 줘 반영하되, **사고피해·옵션도 함께 반영**해 피해 큰 HEV가 깨끗한 차를 무조건
  앞서지 않게 한다. 하이브리드/무DCT만 보는 필터 컬럼을 제공하면 좋다.
- **하이브리드=무DCT**(현대 A6MF2H/TMED 6단 자동, DCT 아님) → 울컥 경고에서 제외하고
  "하이브리드만" 필터를 둔다. 전수 개별검증 결과를 입력으로 쓰면 사고/렌트/옵션이 실측이라
  추천 신뢰도가 높다(encar 스킬 3) 참고).
- **PII 절대 미포함**(차량번호·연락처·딜러명). 링크는 공개 URL만.

## 산출/배포
- `scripts/render.js`(Node, 무의존)로 데이터 JSON → `index.html` 생성.
  `node scripts/render.js data.json > index.html`
- 배포는 선택: `vercel deploy --prod`(정적). 데이터 JSON은 저장소에 커밋하지 않는다.

## 주의
- 입력 데이터는 사용자 기기의 일회성 산출물. HTML도 개인 검토용. 공개 배포 시
  매물 데이터의 약관/DB권/개인정보 이슈를 이용자가 확인할 책임.
