# 헤이딜러(HeyDealer) 중고차 조회 스킬

헤이딜러는 상세 페이지가 SSR/렌더되어 **텍스트 파싱**으로 데이터를 얻는다.
상세 URL: `https://www.heydealer.com/market/cars/<해시ID>`

## 상세 추출 (`scripts/detail.js`)
라벨 기준으로 추출:
- 연식/주행/사고/자차 보험처리(건수·금액)/외부패널·프레임/누유·파손
- 색상, 타이어/틴팅 잔량, 차 키 개수
- **"주요 옵션"은 트림 공통 템플릿이라 신뢰 불가** → 차량별 실제 사양은
  **"기아/현대 출고 정보"(신차 추가옵션)** 목록으로 판정.
- 보증 잔여(엔진/주요 부품 km·개월)

## 목록
마켓 목록(`/` 에서 SPA 필터)은 **브랜드→모델 필터 후 카드가 렌더**된다. 카드 링크
`/market/cars/<해시>` 에서 해시ID를 수집한 뒤 상세를 조회. 단 **로그인 게이트·무한
스크롤**로 전수 자동 열거는 제한적(화면에 렌더된 분량만).

## 한계 / 주의
- 목록 전수 자동화 제한(로그인/무한스크롤). 용도(렌트) 이력 표기가 제한적이라
  성능기록부 확인 권장.
- "단순교환 무사고"는 프레임 무사고(외판 교환·자차보험은 별개)임에 유의.
- 약관·robots.txt 준수, 저빈도 조회. 개인정보 저장 금지.
