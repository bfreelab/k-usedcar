# 헤이딜러(HeyDealer) 중고차 조회 스킬

헤이딜러는 상세 페이지가 SSR/렌더되어 **텍스트 파싱**으로 데이터를 얻는다.
상세 URL: `https://www.heydealer.com/market/cars/<해시ID>`

## 상세 추출 (`scripts/detail.js`)
라벨 기준으로 추출:
- 연식/주행/사고/자차 보험처리(건수·금액)/외부패널·프레임/누유·파손
- 색상, 타이어/틴팅 잔량, 차 키 개수
- **"주요 옵션"은 트림 공통 템플릿이라 신뢰 불가** → 차량별 실제 사양은
  **"기아/현대 출고 정보"(신차 추가옵션)** 목록으로 판정.
- 보증 잔여(엔진/주요 부품 km·개월)

## 목록 — ★헤드(headed) 모드 (헤드리스는 `/market` 404·로그인게이트로 막힘)
헤드리스에서는 목록이 안 뜨지만 **보이는 크롬(headed)** 에서는 동작한다(검증됨).
목록 화면은 `/market`이 아니라 **홈(`/`)에서 SPA 필터**로 전환된다.

절차:
1. 홈에서 **브랜드 클릭**(현대/기아). URL이 `/?brand=<코드>` 로 바뀐다.
2. **모델 클릭** → `/?brand=<코드>&model-group=<코드>`. 가격은 `&max-price=2700` 파라미터로.
3. **스크롤로 카드 로딩** 후, `a[href*="/market/cars/<해시>"]` 링크의 해시ID + 카드
   텍스트(모델·세대(NX4/MX5/TM)·연식·주행·가격·사고배지)를 수집. (`scripts/list.js`)
4. 각 해시ID 상세(`/market/cars/<해시>`)에서 `scripts/detail.js`로 옵션/사고/자차보험 추출.
   ※ "주요 옵션"은 트림 템플릿이라 신뢰 불가 → **"출고 정보"(실제 추가옵션)** 로 판정.

색상은 카드/상세에 잘 안 나오므로 필요 시 상세 본문에서 보강. 용도(렌트) 표기는 제한적.

## 한계 / 주의
- **반드시 headed 모드.** 무한스크롤이라 화면 로딩분까지만(전수는 스크롤 반복 필요).
- "단순교환 무사고"는 프레임 무사고(외판 교환·자차보험은 별개)임에 유의.
- 약관·robots.txt 준수, 저빈도 조회. **개인정보 저장·출력 금지([PRIVACY.md](../../PRIVACY.md)).**
